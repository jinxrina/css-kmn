CSS Fundamentals

Master the building blocks of CSS - selectors, specificity, cascade, and inheritance.

Universal Selector
Selects all elements on the page. Use sparingly for performance.
* { margin: 0; padding: 0; }
Type Selectors
Target elements by their HTML tag name.
h1 { font-size: 2rem; } p { line-height: 1.6; }
Class Selectors
Most commonly used selector for styling components.
.button { padding: 12px 24px; } .card { border-radius: 8px; }
ID Selectors
High specificity selector for unique elements.
#header { position: fixed; } #main-nav { z-index: 100; }
Attribute Selectors
Select elements based on their attributes.
input[type="email"] { border: 2px solid blue; } a[href^="https"] { color: green; }
Pseudo-classes
Style elements based on their state or position.
button:hover { background: #007bff; } li:nth-child(odd) { background: #f9f9f9; }
Specificity Hierarchy
Understanding CSS specificity: Inline > IDs > Classes > Elements
/* Specificity: 0,0,0,1 */ p { color: black; } /* Specificity: 0,0,1,0 */ .text { color: blue; } /* Specificity: 0,1,0,0 */ #main { color: red; }
!important
Override specificity (use sparingly!)
.emergency { color: red !important; }
Inheritance
Some properties inherit from parent elements
body { font-family: Arial; /* Inherited by children */ color: #333; /* Inherited by children */ }
Custom Properties
CSS variables for maintainable code
:root { --primary: #007bff; --spacing: 1rem; } .button { background: var(--primary); }

Typography & Text

Create beautiful, readable text with advanced typography techniques.

font-family
Define font stack with fallbacks
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
font-size
Responsive font sizing with clamp()
font-size: clamp(1rem, 2.5vw, 2rem); font-size: 1.25rem; /* 20px if root is 16px */
font-weight
Control font thickness
font-weight: 300; /* Light */ font-weight: 400; /* Normal */ font-weight: 600; /* Semi-bold */ font-weight: 800; /* Extra-bold */
line-height
Control vertical spacing between lines
line-height: 1.6; /* 160% of font-size */ line-height: 1.5; /* Good for body text */ line-height: 1.2; /* Good for headings */
text-align
Horizontal text alignment
text-align: left | center | right | justify;
text-decoration
Add or remove text decorations
text-decoration: underline wavy red; text-decoration: line-through; text-decoration: none; /* Remove underline */
text-transform
Transform text case
text-transform: uppercase; text-transform: lowercase; text-transform: capitalize;
letter-spacing
Adjust space between characters
letter-spacing: 0.05em; /* Slightly spaced */ letter-spacing: 0.1em; /* More spaced */ letter-spacing: -0.02em; /* Tighter */
@import
Import Google Fonts (slower method)
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;600&display=swap');
@font-face
Define custom fonts
@font-face { font-family: 'MyFont'; src: url('font.woff2') format('woff2'); font-display: swap; }

Layout Systems

Master positioning, floats, and the CSS box model for precise layouts.

box-sizing
Control how width and height are calculated
* { box-sizing: border-box; } /* Recommended */ box-sizing: content-box; /* Default */
margin
Space outside the element
margin: 20px; /* All sides */ margin: 10px 20px; /* Vertical | Horizontal */ margin: 10px 15px 20px 25px; /* Top Right Bottom Left */
padding
Space inside the element
padding: 1rem; /* All sides */ padding: 0.5rem 1rem; /* Vertical | Horizontal */ padding-inline: 1rem; /* Logical property */
position: static
Default positioning - normal document flow
position: static; /* Default value */
position: relative
Positioned relative to its normal position
position: relative; top: 10px; left: 20px;
position: absolute
Positioned relative to nearest positioned ancestor
position: absolute; top: 0; right: 0;
position: fixed
Positioned relative to the viewport
position: fixed; top: 20px; right: 20px;
position: sticky
Switches between relative and fixed based on scroll
position: sticky; top: 0; /* Stick when 0px from top */
display: block
Full width, stacks vertically
display: block; /* div, p, h1-h6 default */
display: inline
Only as wide as content, flows horizontally
display: inline; /* span, a, strong default */
display: inline-block
Flows inline but accepts width/height
display: inline-block; /* Best of both worlds */
display: none
Completely removes element from layout
display: none; /* vs visibility: hidden */

flex-direction
Sets the main axis direction

Aligns items along main axis
justify-content: center; justify-content: space-between; justify-content: space-around; justify-content: space-evenly;

Aligns items along cross axis

Space between flex items

Animations & Transitions

Bring your designs to life with smooth transitions and powerful keyframe animations.

üé™ Animation Playground
transition
Shorthand for all transition properties
transition: all 0.3s ease; transition: opacity 0.5s ease-in-out; transition: transform 0.3s cubic-bezier(0.4, 0, 0.2, 1);
transition-property
Which properties to animate
transition-property: opacity, transform; transition-property: all; /* default */ transition-property: none;
transition-duration
How long the transition takes
transition-duration: 0.3s; transition-duration: 300ms; transition-duration: 0.3s, 0.5s; /* Multiple values */
transition-timing-function
The speed curve of the transition
transition-timing-function: ease; /* default */ transition-timing-function: ease-in-out; transition-timing-function: cubic-bezier(0.25, 0.46, 0.45, 0.94);
@keyframes
Define animation steps
@keyframes slideIn { from { transform: translateX(-100%); } to { transform: translateX(0); } }
animation
Shorthand for all animation properties
animation: slideIn 1s ease-in-out; animation: bounce 2s infinite; animation: fadeIn 0.5s ease-out forwards;
animation-iteration-count
How many times to repeat
animation-iteration-count: 1; /* default */ animation-iteration-count: infinite; animation-iteration-count: 3;
animation-fill-mode
What happens before/after animation
animation-fill-mode: none; /* default */ animation-fill-mode: forwards; /* Keep end state */ animation-fill-mode: backwards; /* Apply start state */ animation-fill-mode: both;
transform: translate
Move elements without affecting layout
transform: translateX(50px); transform: translateY(-20px); transform: translate(10px, 20px); transform: translate3d(10px, 20px, 0);
transform: rotate
Rotate elements around their center
transform: rotate(45deg); transform: rotateX(60deg); transform: rotateY(180deg); transform: rotateZ(90deg);
transform: scale
Resize elements
transform: scale(1.2); /* 120% size */ transform: scaleX(0.8); /* 80% width */ transform: scaleY(1.5); /* 150% height */ transform: scale(1.2, 0.8);
transform: skew
Skew elements along X or Y axis
transform: skew(20deg); transform: skewX(15deg); transform: skewY(10deg);
Multiple Transforms
Combine multiple transformations
transform: translateX(50px) rotate(45deg) scale(1.2); transform: translate3d(10px, 20px, 0) rotateZ(90deg);
transform-origin
Set the pivot point for transformations
transform-origin: center; /* default */ transform-origin: top left; transform-origin: 50% 50%; transform-origin: 100% 0;
CSS Cubic Bezier
Custom easing functions
/* Ease-out-back */ transition: all 0.3s cubic-bezier(0.175, 0.885, 0.32, 1.275); /* Ease-in-out-quart */ transition: all 0.5s cubic-bezier(0.77, 0, 0.175, 1);
Animation Delay
Stagger animations for better UX
.item:nth-child(1) { animation-delay: 0.1s; } .item:nth-child(2) { animation-delay: 0.2s; } .item:nth-child(3) { animation-delay: 0.3s; }
Performance Tips
Optimize animations for 60fps
/* Use transform and opacity for best performance */ .optimized { will-change: transform, opacity; } /* Avoid animating layout properties */ /* DON'T: width, height, margin, padding */

Responsive Design

Create websites that work perfectly on all devices with modern responsive techniques.

Basic Media Query
Apply styles based on screen size
@media (max-width: 768px) { .container { padding: 1rem; } } @media (min-width: 1200px) { .container { max-width: 1140px; } }
Common Breakpoints
Standard responsive breakpoints
/* Mobile First Approach */ @media (min-width: 576px) { /* Small devices */ } @media (min-width: 768px) { /* Medium devices */ } @media (min-width: 992px) { /* Large devices */ } @media (min-width: 1200px) { /* Extra large */ }
Orientation Queries
Target device orientation
@media (orientation: portrait) { /* Styles for portrait mode */ } @media (orientation: landscape) { /* Styles for landscape mode */ }
Feature Queries
Test for specific device capabilities
@media (hover: hover) { /* Styles for devices that can hover */ button:hover { background: blue; } } @media (pointer: coarse) { /* Styles for touch devices */ button { min-height: 44px; } }
Flexible Images
Make images responsive
img { max-width: 100%; height: auto; } /* Modern aspect ratio control */ .image-container { aspect-ratio: 16 / 9; }
Fluid Typography
Typography that scales smoothly
h1 { font-size: clamp(2rem, 5vw, 4rem); } p { font-size: clamp(1rem, 2.5vw, 1.25rem); }
Flexible Layouts
Layouts that adapt to content
.container { width: min(90%, 1200px); margin: 0 auto; } .grid { grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); }
Viewport Units
Units relative to viewport size
/* Viewport Width/Height */ width: 100vw; /* Full viewport width */ height: 100vh; /* Full viewport height */ /* Minimum of vw and vh */ font-size: 5vmin; /* Maximum of vw and vh */ font-size: 5vmax;
CSS Functions
Mathematical functions for responsive design
/* Clamp: min, preferred, max */ font-size: clamp(1rem, 4vw, 2rem); /* Min/Max */ width: min(90%, 1200px); height: max(300px, 50vh); /* Calc */ width: calc(100% - 2rem);
Logical Properties
Direction-aware properties
/* Instead of margin-left/right */ margin-inline: 1rem; /* Instead of margin-top/bottom */ margin-block: 2rem; /* Instead of width */ inline-size: 100%; /* Instead of height */ block-size: 200px;
@container
Style based on container size (modern browsers)
.card-container { container-type: inline-size; } @container (min-width: 400px) { .card { display: flex; gap: 1rem; } }
Intrinsic Web Design
Let content define the layout
.intrinsic-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); gap: 1rem; } .item { min-height: min-content; }

Advanced CSS Techniques

Master advanced CSS features like custom properties, pseudo-elements, and complex selectors.

CSS Variables
Define reusable values with custom properties
:root { --primary-color: #007bff; --spacing-unit: 1rem; --border-radius: 8px; } .button { background: var(--primary-color); padding: var(--spacing-unit); border-radius: var(--border-radius); }
Dynamic Theming
Create theme systems with CSS variables
[data-theme="dark"] { --bg-color: #1a1a1a; --text-color: #ffffff; } [data-theme="light"] { --bg-color: #ffffff; --text-color: #333333; } body { background: var(--bg-color); color: var(--text-color); }
Variable Fallbacks
Provide fallback values for custom properties
.element { /* Fallback to blue if --primary is not defined */ color: var(--primary-color, blue); /* Nested fallbacks */ background: var(--bg, var(--fallback-bg, white)); }
::before & ::after
Create decorative elements without extra HTML
.quote::before { content: """; font-size: 2em; color: #ccc; } .quote::after { content: """; font-size: 2em; color: #ccc; }
::first-letter
Style the first letter of an element
.article::first-letter { font-size: 3em; float: left; line-height: 1; margin-right: 0.1em; font-weight: bold; }
::selection
Style text selection
::selection { background: #007bff; color: white; } .special::selection { background: #ff6b6b; color: white; }
::placeholder
Style input placeholder text
input::placeholder { color: #999; font-style: italic; opacity: 1; /* Firefox */ } input:focus::placeholder { opacity: 0.5; }
:nth-child() Patterns
Advanced nth-child selectors
/* Every 3rd item */ li:nth-child(3n) { background: #f0f0f0; } /* Every 3rd item starting from 2nd */ li:nth-child(3n+2) { color: blue; } /* First 3 items */ li:nth-child(-n+3) { font-weight: bold; } /* All except first 3 */ li:nth-child(n+4) { margin-top: 1rem; }
:has() Selector
Parent selector (modern browsers)
/* Card with an image */ .card:has(img) { border: 2px solid blue; } /* Form with errors */ .form:has(.error) { border-color: red; } /* Article with headings */ .article:has(h2) { margin-bottom: 2rem; }
:is() & :where()
Grouping selectors with different specificity
/* :is() maintains highest specificity */ :is(h1, h2, h3) { margin-top: 0; } /* :where() has 0 specificity */ :where(h1, h2, h3) { margin-bottom: 1rem; } /* Complex grouping */ :is(.card, .panel) :is(h2, h3) { color: var(--heading-color); }
:not() Selector
Exclude elements from selection
/* All buttons except primary */ button:not(.primary) { background: #f8f9fa; } /* All links except external */ a:not([href^="http"]) { color: var(--internal-link-color); } /* Complex exclusions */ .item:not(:first-child):not(:last-child) { border-radius: 0; }
clip-path
Create custom shapes
/* Circle */ clip-path: circle(50% at center); /* Triangle */ clip-path: polygon(50% 0%, 0% 100%, 100% 100%); /* Hexagon */ clip-path: polygon(30% 0%, 70% 0%, 100% 50%, 70% 100%, 30% 100%, 0% 50%); /* Rounded rectangle */ clip-path: inset(10px 20px 30px 40px round 10px);
mask
Hide parts of elements with images or gradients
/* Gradient mask */ mask: linear-gradient(to bottom, black 0%, black 70%, transparent 100%); /* Image mask */ mask: url(mask.svg); /* Radial mask */ mask: radial-gradient(circle, black 50%, transparent 70%);
backdrop-filter
Apply filters to background
.glassmorphism { background: rgba(255, 255, 255, 0.1); backdrop-filter: blur(10px); border: 1px solid rgba(255, 255, 255, 0.2); } .frosted { backdrop-filter: blur(5px) saturate(1.8); }

Modern CSS Features

Cutting-edge CSS features and best practices for modern web development.

CSS Subgrid
Inherit grid tracks from parent grid
.grid-container { display: grid; grid-template-columns: 1fr 2fr 1fr; gap: 1rem; } .grid-item { display: grid; grid-template-rows: subgrid; grid-row: span 3; }
Grid Template Areas
Named grid areas for semantic layouts
.layout { display: grid; grid-template-areas: "header header header" "nav main aside" "footer footer footer"; grid-template-rows: auto 1fr auto; } .header { grid-area: header; } .nav { grid-area: nav; } .main { grid-area: main; }
scroll-behavior
Smooth scrolling behavior
html { scroll-behavior: smooth; } .scroll-container { scroll-behavior: auto; /* Override for specific container */ }
scroll-snap
Create snap points for scrolling
.scroll-container { scroll-snap-type: x mandatory; overflow-x: scroll; } .scroll-item { scroll-snap-align: center; scroll-snap-stop: always; }
overscroll-behavior
Control scroll chaining behavior
.modal { overscroll-behavior: contain; /* Prevent scroll chaining */ } .sidebar { overscroll-behavior-y: contain; }
color-mix()
Mix colors mathematically (modern browsers)
/* Mix 50% red with 50% blue */ background: color-mix(in srgb, red 50%, blue); /* Mix with different percentages */ background: color-mix(in srgb, var(--primary) 80%, white);
oklch() & oklab()
Perceptually uniform color spaces
/* OKLCH: Lightness, Chroma, Hue */ color: oklch(70% 0.25 180); /* OKLAB: Lightness, A, B */ color: oklab(70% -0.3 0.4); /* Better for color manipulation */ --base-color: oklch(60% 0.15 200); --lighter: oklch(80% 0.15 200);
accent-color
Control form control accent color
:root { accent-color: #007bff; } /* Applies to checkboxes, radio buttons, range sliders */ input[type="checkbox"] { accent-color: var(--primary-color); }
CSS Architecture
Organize CSS for maintainability
/* BEM Methodology */ .card { /* Block */ } .card__title { /* Element */ } .card--featured { /* Modifier */ } /* Utility Classes */ .mt-4 { margin-top: 1rem; } .text-center { text-align: center; } .sr-only { /* Screen reader only */ }
Performance Optimization
CSS for better performance
/* Use contain for isolated components */ .component { contain: layout style paint; } /* Optimize animations */ .animated { will-change: transform; } /* Remove will-change after animation */ .animated.complete { will-change: auto; }
Accessibility First
CSS for inclusive design
/* Respect user preferences */ @media (prefers-reduced-motion: reduce) { * { animation-duration: 0.01ms !important; animation-iteration-count: 1 !important; transition-duration: 0.01ms !important; } } /* High contrast support */ @media (prefers-contrast: high) { .button { border: 2px solid; } }
Modern Reset
A modern CSS reset for better defaults
/* Modern CSS Reset */ *, *::before, *::after { box-sizing: border-box; } * { margin: 0; } html, body { height: 100%; } body { line-height: 1.5; -webkit-font-smoothing: antialiased; } img, picture, video, canvas, svg { display: block; max-width: 100%; }
